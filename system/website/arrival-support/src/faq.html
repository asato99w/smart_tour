❓ よくある質問（FAQ）

お客様からよくいただくご質問にお答えします

🛫 どの空港で利用できますか？

+

現在、スワンナプーム国際空港（BKK）とドンムアン国際空港（DMK）でサービスを提供しています。その他の空港についてもお気軽にお問い合わせください。

⏰ 深夜・早朝便でも対応可能ですか？

+

はい、24時間対応しています。深夜便や早朝便でも追加料金なしでサポートいたします。

📱 どのようなアプリをインストールしてもらえますか？

+

Grab（配車アプリ）、LINE、Google翻訳、Google Maps、通貨換算アプリなど、タイ滞在に必要なアプリをインストール・設定いたします。その他ご希望のアプリがあればお申し付けください。

💳 SIMカードの種類は選べますか？

+

はい、滞在期間やデータ使用量に応じて最適なSIMカードをご提案します。主要キャリア（AIS、dtac、True）から選択可能です。

💰 料金の支払い方法は？

+

現金（タイバーツ・日本円）、クレジットカード、PayPay、LINE Payでお支払いいただけます。事前決済も可能です。

📅 いつまでに予約すれば良いですか？

+

できるだけ早めのご予約をお勧めしますが、最低でも到着の24時間前までにご予約ください。緊急の場合はお電話でご相談ください。

🔄 キャンセル・変更は可能ですか？

+

到着の24時間前までは無料でキャンセル・変更が可能です。24時間以内の場合は50%のキャンセル料が発生します。

👥 複数人での利用は可能ですか？

+

はい、グループでのご利用も可能です。3名様以上の場合は割引料金を適用いたします。詳細はお問い合わせください。

🗣️ 日本語でサポートしてもらえますか？

+

はい、すべてのスタッフが日本語対応可能です。タイ語や英語が分からなくても安心してご利用いただけます。

📞 サービス後のサポートはありますか？

+

はい、滞在中に困ったことがあれば24時間LINEでサポートいたします。追加料金はかかりません。

🏥 緊急時の対応はしてもらえますか？

+

病気やケガなどの緊急時には、病院への同行や通訳サポートを行います（別途料金）。まずはLINEまたはお電話でご連絡ください。

🛍️ ショッピングや観光の案内もしてもらえますか？

+

フルサポートプランでは周辺の観光地やショッピングエリアの案内も含まれています。その他のプランでも別途ガイドサービスを提供可能です。

🎒 荷物の預かりサービスはありますか？

+

申し訳ございませんが、荷物の預かりサービスは行っておりません。空港内のロッカーサービスや宿泊先での荷物預かりをご案内いたします。

🚗 レンタカーの手配はできますか？

+

レンタカー会社のご紹介は可能ですが、直接の手配は行っておりません。国際運転免許証の確認や注意事項についてもご案内いたします。

🏪 両替はどこでするのがお得ですか？

+

空港内の両替所よりも市内の両替所の方がレートが良い場合が多いです。お客様のご希望に応じて最適な両替所をご案内いたします。

🌡️ タイの気候について教えてください

+

タイは年間を通して暑く、3月〜5月が最も暑い時期です。6月〜10月は雨季、11月〜2月は乾季で比較的過ごしやすくなります。服装や持ち物についてもアドバイスいたします。

💉 ワクチン接種は必要ですか？

+

現在、タイ入国に必要なワクチン接種はありません。ただし、最新の入国要件については外務省や大使館の情報をご確認ください。

他にご質問がございますか？

お気軽にお問い合わせください。24時間以内にご回答いたします。
